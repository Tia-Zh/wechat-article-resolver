# 给 AI 的安装与使用指令

把下面这段话发给支持本机命令行/文件操作的 AI，例如 Codex、Claude Code、Cursor Agent 等。

```text
请安装这个 GitHub 仓库里的微信公众号文章解析工具：

1. clone 仓库到本机合适目录。
2. 进入仓库目录。
3. 在 Windows 上运行：powershell -ExecutionPolicy Bypass -File .\install.ps1
4. 以后当我给你微信公众号文章链接时，请先尝试用你自己的网页读取能力读取。
5. 如果你能完整读取正文，就直接继续我原本要求你做的任务。
6. 如果你读不到正文、只读到按钮/导航/异常页，或正文明显不完整，请运行本工具：
   .\resolve-wechat.cmd "公众号链接"
7. 工具会输出 Markdown。请读取输出的 Markdown 正文，再继续我原本要求你做的任务。
8. 解析正文只是前置步骤，后续仍按我原来的要求执行。
```

注意：普通网页聊天 AI 如果不能访问本机命令行，就无法安装或调用这个工具。
